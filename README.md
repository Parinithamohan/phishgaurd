PhishGaurd
Solution: AI-based browser extension that
evaluates every URL visited.
How it works:
1.Extension captures URLs in real-time.
2.Features extracted (domain length, keywords,
suspicious patterns).
3.AI model predicts phishing likelihood.
4.Safety score returned: Green (Safe) / Yellow
(Suspicious) / Red (Dangerous)
Key Value: Protects users proactively
without disrupting browsing.
